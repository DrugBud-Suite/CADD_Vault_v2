About CADD Vault

CADD Vault is a comprehensive repository of computational chemistry software packages, libraries, and resources. Our goal is to help researchers, students, and professionals discover tools that can accelerate their work in drug discovery, molecular modeling, and other computational chemistry domains.

The database includes information about GitHub repositories, citations, documentation, and user ratings to help you make informed decisions about which tools to use for your specific needs.
